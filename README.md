# Graduation-Models (Rafiq AI Models)

## Overview

This repository contains the machine learning and deep learning models developed for **Rafiq**, an AI-powered mental health support system created as our graduation project.

Rafiq is designed to promote mental health awareness by providing early psychological assessment, emotional analysis, and AI-assisted support through intelligent predictive models and natural language processing. The project aims to encourage early intervention while helping reduce the stigma surrounding mental health.

---

## Repository Structure

The repository contains both the production models integrated into the final system and experimental models developed during the research phase of the project.

### Production Models

The following models constitute the core AI components of the final version of Rafiq.

#### General Test Model

The General Test Model serves as the initial mental health assessment. Based on users' responses to a comprehensive questionnaire, it identifies potential psychological conditions and determines whether further specialized assessment is recommended.

#### Depression Model

The Depression Model provides a dedicated assessment for depressive symptoms. It performs a more focused evaluation following the initial screening and was optimized using supervised machine learning techniques.

#### Journal Models

The Journal Models analyze users' journal entries using Natural Language Processing (NLP). These models identify emotional patterns, monitor changes over time, and generate insights that contribute to continuous mental health tracking.

#### Chat Bot

The Chat Bot is an AI-powered conversational assistant that provides supportive interactions, understands user input, and delivers context-aware responses. It is designed to complement professional mental health care rather than replace it.

---

## Research and Experimental Models

The following models were developed as part of the research and experimentation process but are not part of the final production pipeline.

### Anxiety Model

An experimental model developed for anxiety detection. Multiple preprocessing strategies and classification approaches were explored to evaluate its effectiveness and potential future integration.

### Journal V

Journal V was developed as an experimental extension of the journaling feature to support **voice-based journaling**. Instead of requiring users to write their thoughts, this prototype leveraged **OpenAI Whisper** to transcribe spoken journal entries into text, allowing users to express themselves more naturally through speech.
